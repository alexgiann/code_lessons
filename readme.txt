Generator

This generator (nth) returns every number starting from number s incrementing by number n. 

Installation

Copy this file to the python path and import the module :

from nth import nth 

Usage

>>>  from nth import nth
>>>  odds = nth(2, 1)
>>>  next(odds)
>>> 1
>>> next(odds)
>>> 3

>>> tens = nth(0, 10)
>>> next(tens)
>>> 0
>>> next(tens)
>>> 10
